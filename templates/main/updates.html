Первый релиз — дипломная версия

  • AI-оптимизация текстов через Ollama (Llama 3.1:8b)
  • Публикация в Threads через официальный API
  • Планировщик постов с расписанием
  • Управление кампаниями (BASIC / PREMIUM / ENTERPRISE)

  • Мониторинг упоминаний и ответов
  • Трендовые темы из базы 21 000+ постов Threads
  • OAuth-подключение Threads-аккаунтов
  • Интерфейс на RU / EN / KZ
  • Тёмная и светлая темы
  • Cloudflare Tunnel для публичного HTTPS
